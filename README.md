# 3D-Models
3D-Models is a AI Powered Website made using vite and three.js.

## Tech Stack

**Client:** React JS

  
## Run Locally

Clone the project

```bash
  git clone https://github.com/poudyalamit/3D-Models
```

Go to the project directory

```bash
  cd 3D-Models
```

Install dependencies

```bash
  cd client/
  npm install
```

Start the Client

```bash
  //open now terminal
  cd client
  npm run dev
```
